⚡️ 刷写 Fydetab Duo

以下是在 Fydetab Duo 上刷入新操作系统的详细步骤指南：

准备工作​

在您开始任何操作之前，请确保您的设备已准备就绪，并且手边备齐了所有必需工具。接下来我们将引导您完成进入Loader模式、检查设备状态以及刷写新操作系统镜像的完整流程。

第一步：准备您的设备​

  1. 确保设备已完全关机
    在将 Fydetab Duo 连接到您的电脑之前，请确保设备已完全关机。如果设备自动启动，请再次关机，以避免意外的操作。

  2. 将Fydetab Duo 连接到您的电脑 将 Fydetab Duo 使用 USB Type-C 数据线连接到你打算用来刷写系统的电脑。

注意

由于工厂供应商的疏忽，Fydetab Duo 所附带的 Type-C 数据线不支持数据传输，因此你无法使用它进入加载模式。请使用其它可处理数据传输的 Type-C 数据线。

第二步：启动到 Loader/Maskrom 模式​

根据以下的步骤来进入 Loader 模式，如有必要，也可以进入 Maskrom 模式：

  1. 长按音量加键（vol+）
    按下并按住您设备上的音量加键（vol+）。

  2. 轻按电源按钮
    在持续按住 vol+ 按钮的同时，轻按下电源按钮 0.5 到 1 秒。

  3. 同时松开两个按钮
    这会使得设备进入 Loader 模式。

提示

如果同时送开两个按钮不起效，可尝试先松开电源按键，再松开音量加键（vol+）

  1. (如果使用Windows) 切换到 Maskrom 模式
    如有必要，使用 RockdevTool 切换到 Maskrom 模式（查阅下方的 Windows 选项卡以获得更多信息）。

第三步：检查设备状态​

如果一切顺利，你的刷写设备此时应该能识别到 Fydetab Duo。在 Loader 模式下，屏幕可能会显示一个特定图示，提示设备已经进入 Loader Mode。

注意

当设备的 firmware 是初版时，设备的 LED 灯将不会亮起，屏幕也将保持全黑状态，这是正常的。

第四步：刷写镜像​

  • 🐧 Linux (Debian/Ubuntu)
  • 🍏 macOS
  • 🖥️ Windows
注意

该 Linux 升级工具是瑞芯微（Rockchip）专有的解决方案，用于将镜像文件烧录到 SPI、eMMC、SD 卡等多种存储设备上。与开源的 rkdeveloptool 不同，此工具仅以二进制可执行文件的形式分发。

  1. 下载升级工具
    通过链接下载原始格式的 upgrade_tool。

  2. 赋予文件可执行权限
    打开终端，运行以下命令以赋予文件可执行权限：

    sudo chmod +x /path/to/upgrade_tool

  3. 烧录镜像
    解压下载的镜像文件，然后使用以下命令进行烧录：

    sudo ./path/to/upgrade_tool uf /path/to/unarchived/image-name.img

Requires macOS > 10.15

  1. 下载升级工具
    在此处下载升级工具：下载链接。

  2. 解压镜像文件
    使用你偏好的工具来解压已下载的镜像文件。

  3. 刷写镜像
    打开终端，并执行以下命令：

    chmod +x upgrade_tool
    sudo ./upgrade_tool uf <更新镜像的文件名>.img

  1. 安装 Rockchip 驱动
    下载并安装 Rockchip 驱动。

  2. 下载 RKDevTool
    在此处获取 RKDevTool：下载链接。

  3. 刷写镜像
    使用 RKDevTool 的图形界面来刷写镜像。详细操作说明，请参阅 此指南。
